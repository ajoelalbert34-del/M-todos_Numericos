¿Qué es el Método de Simpson 3/8?

Es un método de integración numérica que aproxima el valor de una integral definida mediante un polinomio interpolante de grado 3 (cúbico).

El método de Simpson 3/8 es exacto para polinomios de grado ≤ 3, ya que ajusta perfectamente la curvatura cúbica de la función.

Simpson 3/8 Simple

Usa exactamente 4 puntos: x₀, x₁, x₂, x₃.

n = 3 subintervalos.

Coeficientes Cᵢ: 1, 3, 3, 1.

Simpson 3/8 Compuesto

Divide el intervalo en n subintervalos.

n debe ser múltiplo de 3.

Genera n + 1 puntos.

Los coeficientes Cᵢ siguen el patrón 1, 3, 3, 2, 3, 3, 2, ..., 3, 3, 1.

Error del Método

Si se conoce el valor exacto o de referencia de la integral, se pueden calcular los siguientes errores:

Error absoluto

Error relativo

Error porcentual

Si Iexacta = 0, el error relativo y porcentual no se pueden calcular. Se muestra solo el error absoluto.
