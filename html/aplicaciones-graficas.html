Métodos Numéricos

Simuladores visuales del Método de Simpson 3/8

Cada aplicación interpreta el área bajo una curva como una magnitud acumulada: volumen de agua, carga contaminante o volumen excavado. La lógica de Simpson 3/8 se mantiene aislada del simulador general.

Crecida de río

Integra el caudal Q(t) para estimar el volumen acumulado de agua en un embalse.

Q(t)m³Caudal → Volumen

Descarga contaminante

Integra la tasa C(t) para estimar la carga contaminante acumulada en un río.

C(t)kgTasa → Carga

Volumen excavado

Integra áreas transversales A(x) para estimar el volumen de excavación por secciones.

A(x)m³Área → Volumen
